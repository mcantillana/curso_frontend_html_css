Desafío tecnológico para emprendedores

Santiago, 17 de agosto de 2015.- Con el objetivo de potenciar y fortalecer emprendimientos basados en tecnología Near Field Communication (NFC), Claro y el CIET de la Universidad Adolfo Ibáñez, con el apoyo de CORFO, lanzaron el concurso llamado NFC Challenge.

Se trata de un nuevo programa que tiene la finalidad de entregar conocimientos, capacitación y acompañamiento durante el proceso de levantamiento de ideas o proyectos y soluciones basadas en NFC con potencial de convertirse en soluciones de alto impacto.

NFC es una tecnología que permite el intercambio de datos a corta distancia entre dispositivos tales como smartphones, tarjetas con chip, tags, entre otros. Sus usos pueden simplificar actividades, tales como; control de acceso, fidelización de clientes, pago móvil, o marketing móvil. Leer mas..

Los lenguajes de programación que deberías aprender para empezar a programar

Es cierto, todo depende de lo que quieras programar. Para páginas web, por ejemplo, deberías aprender HTML, CSS, JavaScript y PHP para la interactividad. Si tu objetivo es, en cambio, desarrollar aplicaciones móviles, lo que deberías aprender, para empezar, sería Objective-C para iOS o Java para Android.

Sin embargo, todo tiene un inicio. Acá te dejamos con algunos de los lenguajes de programación más recomendados para principiantes.

  • C: Programa de manera más eficiente
  • C es uno de los lenguajes de programación más usados, ya que funciona a nivel de máquina, por lo que sirve para entender cómo interactúa un programa con la capa de hardware.
  • Java: Uno de los lenguajes más útiles de aprender
  • Python: Fácil y divertido
  • Javascript: Para dar el salto a la programación web

No te confundas, Javascript no tiene mucha relación con Java. Su puesta en marcha es relativamente sencilla, ya que prácticamente está integrado en todos los navegadores. JavaScript porque es bastante flexible en sintaxis, puedes ver inmediatamente los resultados de lo que has hecho y no es necesario aprender muchas herramientas. Si quieres programar elementos interactivos para una página web, JavaScript es un imprescindible. Leer mas..
